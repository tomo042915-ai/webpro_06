# レポート作成タスクリスト 

## 準備段階 
- [x] Manabaから`example.md`をダウンロードする。
- [ ] レポートの主題/テーマを決定する。
- [ ] レポートの構成案（見出し構成）を検討する。

## データ収集・調査段階 
- [ ] レポートに必要な文献・情報を収集する。
- [ ] 収集した情報を整理し、レポートの構成案に割り当てる。

## 執筆段階 
- [ ] `example.md`を参考に、メインファイル（例: `report.md`）を作成する。
- [ ] **大見出し**（タイトル）を作成する。
- [ ] **中見出し**（節）を作成し、本文を記述する。
  - [ ] はじめに（目的・背景）を記述する。
  - [ ] 本論（調査結果や考察など）を記述する。
  - [ ] おわりに（まとめ・結論）を記述する。
- [ ] 必要に応じて**小見出し**（項）を作成し、本文を詳細化する。
- [ ] 図表や画像（もしあれば）を挿入する。
- [ ] 参考文献リストを作成する。
